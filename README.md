This is a production-ready starter for a single-deploy fullstack app:
- Backend: Express (serves API + built frontend)
- Frontend: React (Vite)
- Azure integration: supports managed identity / Key Vault / Blob Storage SAS generation
- Deployable to Azure App Service

## Run locally (dev)
- Backend: `cd backend && npm install && npm run dev`
- Frontend: `cd frontend && npm install && npm run dev`
- To build production frontend: `cd frontend && npm run build`
- To run production server: `cd backend && npm start`
